How to Run SkillChain Backend

1. Install Node.js (LTS version)

2. Clone repository
git clone https://github.com/YOURUSERNAME/skillchain.git
cd skillchain

3. Install dependencies
npm install

4. Create .env file inside the folder

Add:
MNEMONIC=your 25 word algorand mnemonic

5. Start server
node server.js

6. Open browser
http://localhost:3000
